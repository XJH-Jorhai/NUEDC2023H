# NUEDC2023H

这是一个针对中国全国大学生电子设计竞赛 2023 年 H 题的完整题解与开源项目。

本项目计划整理题目解法方案、自行设计硬件内容，以及题解所用的 STM32 单片机程序，方便复现、学习和二次改进。

## 项目结构

```text
NUEDC2023H/
├── docs/       # 题目原文、题解文档、设计资料、报告 LaTeX 模板
├── hardware/   # 模块连接方式、硬件原理图设计等硬件资料
└── software/   # 基于 FK407-template 二次构建的 STM32F407 程序
```

## 内容规划

### `docs/`

- 题目原文
- 题目题解的 Markdown 文档
- 设计相关资料，例如所用模块的说明书、数据手册等
- 设计报告及 LaTeX 源码

### `hardware/`

- 模块连接方式
- 硬件原理图设计
- 其他用于复现硬件方案的资料

### `software/`

- 基于 [XJH-Jorhai/FK407-template](https://github.com/XJH-Jorhai/FK407-template) 二次构建的 STM32F407 软件
- 项目相关驱动、应用逻辑与构建说明
